# 🍉 西瓜宇宙 — 系统总览

> 版本：v2.1 | 日期：2026-02-14
> 状态：Charles 确认

## 产品定位

**用专注时间驱动的西瓜养成收藏游戏。**

专注是核心，瓜棚是中转站，农场是目标，图鉴是成就。

---

## 三大系统

| 系统 | 一句话定位 | 核心动作 |
|------|-----------|---------|
| ⏱️ 番茄时钟 | 专注工具，一切的起点 | 专注 → 获得收获物 |
| 🏠 瓜棚 | 仓库 + 工坊，资源中转站 | 合成 → 切瓜 → 获得种子和道具 |
| 🌱 农场 | 种植园 + 图鉴，最终目标 | 种植 → 培育 → 收获品种 → 收藏 |

---

## 核心循环

```
⏱️ 番茄时钟                🏠 瓜棚                    🌱 农场
┌──────────┐          ┌──────────────┐          ┌──────────────┐
│          │  收获物   │              │  种子     │              │
│  专 注   │ -------> │  存放 & 合成  │ -------> │  种植 & 培育  │
│          │          │  切瓜 & 管理  │  道具     │  收获 & 收藏  │
└──────────┘          └──────────────┘          └──────────────┘
     ↑                                                │
     └──── 里程碑奖励（新主题/新背景音）←── 图鉴收集 ──┘
```

**用户视角一句话：** 专注攒瓜 → 合成大瓜 → 切开得种子 → 种到农场看它长大 → 收获进图鉴

---

## 系统间的资源流动

| 资源 | 产出地 | 存放地 | 消耗地 | 说明 |
|------|--------|--------|--------|------|
| 收获物（小芽~大西瓜） | 番茄时钟 | 瓜棚 | 瓜棚（合成/切瓜） | 专注的直接产出 |
| 神秘种子 | 瓜棚（切瓜） | 瓜棚 | 农场（种植） | 进入农场的唯一门票 |
| 道具 | 瓜棚（切瓜） | 瓜棚 | 农场（使用） | 只在农场使用 |
| 特色品种西瓜 | 农场（成熟收获） | 图鉴 | — | 收藏品，解锁成就 |

---

## 概念清单（防混淆）

| 概念 | 是什么 | 不是什么 |
|------|--------|---------|
| **收获物** | 专注产出的 6 种东西（小芽/幼苗/小花/小瓜/大西瓜/金西瓜） | 不是种子，不能直接种到农场 |
| **神秘种子** | 切开大西瓜/金西瓜后蹦出来的东西，品种未知 | 不是收获物，不是专注直接产出 |
| **道具** | 切瓜时偶尔蹦出的辅助物品，只在农场使用 | 不是收获物，不是种子 |
| **特色品种** | 农场里种子长大成熟后揭晓的西瓜品种 | 不是瓜棚里的大西瓜 |
